# 🟥 BAM Parser

## 🔹 Introduction
**BAM-parser** is a tool made by spokwn, its purpose is to parse the BAM registry key.
BAM (Background Activity Moderator) is a Windows service that tracks which apps run in the background. They provide **evidence of execution** in forensics.

## 🎯 What it reveals
- **Background programs** (Program path)
- **Last execution**  (Last execution timestamp)
- **Performs digital signature checks** (Shows if the program is signed, not signed or its a cheat signature)
- **Checks for replaces using journal for every file**
