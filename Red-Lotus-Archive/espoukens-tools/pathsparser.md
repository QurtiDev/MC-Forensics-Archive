# 🟦  PathsParser

## 🔹 Introduction
**PathsParser** is a tool made by spokwn that retrieves data about file paths found in **.txt** files, its purpose is to check digital signatures for each file.

## 🎯 What it reveals
- **Digital signatures** (reports "Signed" or "Not signed")
- **Detects specific digital signatures** (checks for known Cheat signatures)
- **Deleted files** (reports "Deleted" if the file is not found)
- **Checks for replacements using journal**

## ⚙️  How to use
- Place the `.txt` file in the same folder as the program, make sure the file is named `search results.txt`, `paths.txt`, or `p.txt`.
- Run the program.
- Choose if you want to use spokwns rules, use your own rules, check for replacements, scan DLLs only.

Download link: https://github.com/spokwn/PathsParser
