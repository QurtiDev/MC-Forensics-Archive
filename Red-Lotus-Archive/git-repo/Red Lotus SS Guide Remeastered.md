This new **Red Lotus SS Guide Remastered**, accessible via a dedicated website, represents a complete reorganization and consolidation of our fundamental methodologies. It integrates the core information from all previous Red Lotus guides, enriched with details from internal video tutorials, notes, and other archived resources, creating a single, comprehensive source of learning.



**What Does This Guide Cover?**

This guide is designed to be a structured resource covering the essential aspects of ethical and effective ScreenSharing, including:

*   **Foundational Concepts:** Understanding the "why" and "how" of hack checks, the importance of context (instance vs. possession), and the Red Lotus principles guiding our practice (safety, privacy, integrity).
*   **Technical Fundamentals:** Key Windows concepts like NTFS, FAT32, journaling, timestamps ($MFT, $UsnJrnl, $LogFile, ADS, attributes).
*   **Artifact Analysis:** Detailed explanations and analysis techniques for common Windows artifacts (Prefetch, Temp, Recent Items, Recycle Bin, Registry, Event Logs).
*   **Tool Usage:** Guidance on using essential manual tools (System Informer, Journal parsers, Registry Explorer, Nirsoft utilities, etc.) and understanding the role and limitations of automated scanners.
*   **Game-Specific Context:** Focused analysis for environments like Minecraft (Forge mods, Javaedit, etc.).
*   **Input Analysis:** Techniques for investigating mouse software, macros, and debounce time.
*   **Bypass Techniques & Detections:** Comprehensive coverage of common bypass methods (obfuscation, artifact manipulation, clearing, permission changes, memory injection techniques, etc.) and the corresponding detection strategies.
*   **Advanced Forensics:** Introduction to analyzing artifacts like VSS, $INDX, memory dumps, and using tools like Velociraptor and YARA in the context of SS.
*   **Specific Tool Guides:** Information on utilities developed by community members like Spok, Rancio, Echo, and relevant PowerShell scripts.
*   **Cross-Platform Basics:** Introductory guidance for Linux and MacOS ScreenSharing.
*   **Ban Evasion:** Understanding and detecting alternate accounts and evasion tactics.
