### ✨ What's New?

*   **🚀 Massive Performance Overhaul**
> Complete rewrite of the scanning engine.

*   **💾 Multi-Drive Scanning (A-Z)**
> The tool now scans **ALL fixed drives** connected to the system, not just C:. This includes both the instant MFT scan and the **USN Journal Forensics**.

*   **🔍 Recursive Directory & MFT Integration**
> The tool now directly parse the Master File Table (MFT) for instant results across all drives, while maintaining deep recursive scanning for complex folder structures.

> Added detection for **Lunar, Badlion, Feather, Meteor, Wurst, Aristois, Impact, LabyMod, LiquidBounce Prism, Modrinth, ATLauncher, Technic, SKlauncher, PolyMC, CurseForge, MultiMC**, and many more.

*   **🖱️ Right-Click Actions**
> You can now **Right-Click** on any found account to copy the Username or Discord ID directly to your clipboard.

*   **🆔 UUID Auto-Resolver**
> The tool automatically fetches real usernames from Mojang's API for any UUIDs found in logs.
