<================>
* `reg query` command can query all values from the registry based on what path you give, by applying `/s` it will query all keys in BAM
> Use for whatever purposes *you can also extract it by` reg query "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\bam\state\usersettings" /s >> BAMResult.txt`*
## The Command:
`reg query "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\bam\state\usersettings" /s`
