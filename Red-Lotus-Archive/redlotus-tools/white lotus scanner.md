**🧪 The Scanner**
- `WhiteLotus.exe` is written in **Python** and compiled with **py2exe**.
- Requires a **list of file paths** to scan. Preferred input: **`.csv`** (you can switch to **`.txt`** in the selection menu).
- ⚠️ Limitation (free build): **ASCII-only** recognition (UTF-8). The scanner will **break on Unicode** characters.
- Ensure **every path is in quotes** (e.g., `"C:\path\to\file.exe"`). Use an online bulk-quoter or Notepad’s **Edit All** to add quotes quickly.

**▶️ Usage**
1) Run `WhiteLotus.exe` as **Administrator** (or elevate via CMD).  
2) Select a **.txt / .csv** file containing the target file paths.  
3) Results are written to a **`.csv`** — viewable with **Timeline Explorer**.  

**🛠️ Paths Creation**
- Many ways to build your **.csv / .txt** list. Example: **Everything** → *Export*.  
- Check other Red Lotus tools/methods on Discord for path acquisition tips.
