**⚙️ How to run**
- Recommended: launch the `.exe` from **Command Prompt / PowerShell (Run as Admin)**.
- On start, you’ll be asked for a folder path to load files from.  
  Default target (for Minecraft server lists):  
  `C:\Users\%USERNAME%\AppData\Roaming\.minecraft`
- Optional: enter a **filter pattern** (e.g. `.txt`, `.exe`, `.dat`, or a prefix like `f_0`).  
  Leave blank to list **all** files.

**📁 File Listing**
- Shows all files in the chosen directory that match your filter.

**🧭 Selection Mode**
- Use **↑ / ↓** to navigate, **Enter** to open a file.

**🧾 Content View (Hex Dump)**
- A loading screen appears while the file is parsed (press any key if it seems to hang).
- Scroll with **arrow keys** or **PageUp / PageDown**.
- Press **`/`** to search across the entire hex dump.

**🔎 Search Results**
- Matching lines are listed; navigate with **↑ / ↓** or **PageUp / PageDown**.
- Press **Enter** on a result to jump to that line in the hex view.
- Press **b** to go back.

**🚪 Exit**
- Press **q** at any time to quit.
