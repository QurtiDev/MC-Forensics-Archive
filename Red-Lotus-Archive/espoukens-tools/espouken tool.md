# 🛠️ Espouken Tool

## 🔸 Introduction
**espouken** tool is made by spokwn, it's a very useful multi-purpose tool.

## ⭐ Features
- **Scans for time changes**
- **Scans for unicode strings** (Diagtrack, DPS, and Binary files)
- **Scans MC alts** (Very useful for alt checks)
- **Shows HWID**
- **Shows System boot time** 
- **Shows services** (Shows the status and start time of services)

Download link: https://github.com/spokwn/Tool
