> https://github.com/RedLotus-Development/White-Lotus-Scanner
```
WhiteLotus.exe is coded in Python, compiled using PY2EXE.

Executable requires a list of file paths in order to conduct its scanning operation. Preferred paths file extension is .csv, however in the selection menu you can switch it to a .txt.

As a free utility the scanner is limited to ASCII characters recognition, UTF-8 encoding. It will break if a Unicode character is detected.

Make sure all of the files provided are in quotes. You can either use online resources to add quotes to the paths or use notepad's default "Edit All" function.
```
