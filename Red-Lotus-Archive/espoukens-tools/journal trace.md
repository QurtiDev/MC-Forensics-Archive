# 📖  JournalTrace

## 🔸 Intoduction
**JournalTrace** is a tool made by spokwn, made to parse the USN Journal entries.
The USN Journal provides logs of all changes made to files on a **NTFS** volume.

## 🎯 What it reveals
- **The USN** (Update Sequence Number)
- **File name**
- **Date of modification**
- **Reason** (Why the record was written)

## ⚙️ How to use
- Run the program
- Click Drive -> Select
- Choose which Drive you want to parse and then select it
- Click Drive -> Scan to scan the drive
- Go to Layout and choose a way to view the journal

Download link: https://github.com/spokwn/JournalTrace
