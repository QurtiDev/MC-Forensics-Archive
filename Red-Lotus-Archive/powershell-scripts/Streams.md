A script that checks streams from the current path, and you can choose if subdirectories too.
To use it just open cmd as admin, go to the initial path you want to check streams on, and then choose if you want subdirectories, and also how many, after choosing, wait for it to parse.

***Execute in Command Prompt as Admin (One Command for PowerShell 5+)***
```ps
powershell Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass && powershell Invoke-Expression (Invoke-RestMethod https://raw.githubusercontent.com/spokwn/powershells/refs/heads/main/Streams.ps1)
```
